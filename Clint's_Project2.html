Lesson 1: The Basics of the Web and HTML

How the Web Works

The Web is a giant network of devices (ex: laptops, PC's, servers, etc.) that can talk to each other via an Internet connection from the device to an ISP. To connect from one device to another the Web uses HTTP (Hypertext Transfer Protocol) protocol to make this connection.

EXAMPLE:

When you type www.udacity.com your computer sends HTTP protocol via the Internet connection to the Udacity web server. The Udacity web server then communicates back to your computer using HTTP to send the Udacity homepage.

HTML

HTML or Hypertext Markup Languageis the coding language that is the base of any website.

HTML is what shows the visual properties of a website. Whether it's plain text, images, or video, HTML is used to code these properties into the website. For a more detailed description view the following link: Wikipedia-HTML

HTML code uses a structure of HTML elements. The different elements are what define the website you build.

Lesson 2: Creating a Structured Document with HTML

Inline or Block....What's the Difference?

HTML elements are considered either Inline or Block elements. Inline elements usually only displayed without line breaks, such as Underline or Bold elements. Block elements normally contain multiple lines of information. Everything within a block elements is contained in an invisible block. An example of a block element is <div>. This element is used in CSS styling, and when used everything within this tag would use the associated CSS associated styling for that <div> element.

Tree Structure

HTML follows a tree like structure. The structure can start with a container element, such as <div> and within it can contain multiple other elements. There can be many levels of nested elements within one overall element.

Keep Code Clean and Easy to Read

When creating HTML code it is very important to keep it clean and easy to read. The person writing the HTML code initially will likely not be the only person to ever read it again. For this reason, it is a good idea to indent nested elements and keep clear comments where needed.

Lesson 3: Adding CSS Style to HTML Structure

What is CSS?

CSS stands for "Cascading Style Sheets". CSS is the coding language that determines how the HTML code is displayed and styled.

In my own mind, I think of CSS as defining a CSS variable in HTML and then assigning that variable a set of CSS style attributes. Where ever that CSS "variable" is used within HTML, the corresponding elements and its contents use the CSS styling associated.

The following link is a great summary and quick reference from Udacity on CSS. CSS Summary and Reference

Avoid Repetition in Styling

The major benefit of CSS is of not having to duplicate styling multiple times in HTML code. CSS allows for greater efficiency and saves a lot of time for programmers.

CSS coding can be used three different ways.

  • HTML code is linked to a separate CSS file, which is most common, especially for more complicated projects.
  • Within a <style> element within the <head> element at top of HTML code. This method could be used for a smaller project.
  • Write the styling code within the HTML code. This is least preferred method because it creates a lot of repetition and can lead to mistakes and headaches when debugging.

What is Programming?

Programming is a precise sequence of steps to tell the computer or program what to do, and is the core of computer science.

A computer is useless without programming. Computers are not smart themselves, but it's the programming provided by a human that makes them smart.

Programming Languages

A programming language is the language used to write computer programs.In natural speaking languages there is too much ambiguity that can often lead to confusion. Programming languages need clear, concise instructions to remove any confusion.

The building blocks of programming languages are the syntax and semantic rules.

  • Syntax: The set of rules of the language structure or form.
  • Semantics: Defines the meaning of the language used.

There are many different programming languages, for example Python, which is what we are using in this nanodegree. Below are a few other well known programming languages, but there are many more.

  • Java
  • JavaScript
  • PHP
  • SQL
Interpreter

An interpreter is the "middle man" if you will, between the high-level programming language and the lower-level machine language. When we write code in Python it is the Python interpreter program that translates our code into a language that the machine understands.

Lesson 2: Variables and Strings

What is a Variable?

A variable gives the programmer a way to assign a name to commonly used value in the program. Variables can change or update throughout the programming as needed.

EXAMPLE:   value = 2,   print value * 3 >>> 6

How are Variables Useful in Programming?

Variables reduce the amount of code to be typed in a program and improve the readability of others reading the code. A variable takes the place of a value, and can be used multiple times throughout the programming. Throughout the program the value of a variable can change, hence the name variable.

Difference between number and string values

In Python, knowing the difference between a number and a string is very important. Below you can see the differences:

  • A string value should be surrounded by double quotes or single quotes. Even numbers can be made into a string so they can be concatenated with words in a string.

    EXAMPLE: "1000", 'dog', etc.

  • A number value displays a number with no quotation.

    EXAMPLE: 1000, 500, etc.

Lesson 3: Input >>> Function >>> Output

What is a Function (or Procedure)?

A function takes input, performs instructions on that input, and produces the desired output. A simple example would be a function called sum. In this function you could provide a defined amount of input and the function take all the input values and output the sum of all.

Why are Functions so beneficial?

Once functions are created, they can be reused throughout the entire program. This saves a lot of time and effort because you do not have to duplicate so much coding. Once the function is built you just call it into action to get the output needed.

Function format

Functions require a specific format. You first have to define the function and that begins with def. After def you name your function, provide your parameters in parentheses, and end with a semi-colon.

EXAMPLE: def sum(a,b,c):

The body of the function contains the instructions for the function. The body can contain assignments, loops, IF statements, etc. Once all of the instructions are complete the return statement is what is used to produce the output of the function.

EXAMPLE:
def  sum(a,b,c):
x = a + b + c
return x

print  sum(1,3,5) >>> 9

Lesson 4: Decisions and Repetition: If and While

Making Decisions

To make decisions within our functions we have to create test expressions using operators. These operators include: <, >, ==, !=, etc.

The one difference seen is ==. The double equal sign is used for equal, whereas the normal = sign we're accustomed to seeing actually means assignment in Python.

IF Statement & While Loop

The IF statement is used to control which code gets executed when. The IF statement can also include the ELSE clause, which gives an alternative path if the test expression is not TRUE. The IF statement will run only 0 or 1 times.

The WHILE Loop lets you write code that does the same thing over and over. The WHILE loop continues running the loop as long as the <block> is TRUE. Once it is hits FALSE it breaks the loop and continues to the next line of code.

Lesson 5: Structured Data: Lists and For Loops

Structured Data
Strings and Lists
  • Strings are a sequence of characters and we can operate on sub-sequences of the string.
  • Lists are a sequence of anything. Elements in the lists can be strings, numbers, or other lists. Lists are more powerful than strings.
Mutability

The ability to mutate means that we can change the value of a list once we create it, and mutating allows you to change list elements while keeping the same list.

The Difference between Append and "+"

Append is a method used with a list to add a new element(s) to an existing list, and does not create a new list.

The plus ("+") operator is similar to concatenation with strings, and does not mutate the lists, but combines two lists to create a new list.

For Loops

While you could loop through every element of a list with a WHILE loop it could be more complicated. The FOR Loop in Python reduces the work and runs more efficiently than WHILE.

What is Objected Oriented Programming?

Object Oriented Programming (OOP) is a concept of computer programming that utilizes "objects", which are data structures containing variables and methods.

OOP is built on classes, and instances of those classes are objects. This structure is more efficient due to less code and is consistent with the natural way of human thinking, thereby easier to follow and understand. Therefore OOP is ideal for larger, complex programming projects.

OOP and its concept of using classes and objects can be relatable to HTML and CSS. Classes and CSS are similar in that you create them once with the attributes needed, and instances of them, in the form of objects or HTML, can be used in code more efficiently.

Below is a list of terminology associated with OOP, along with their definitions.

Vocabulary Words & Definitions

Class
  • The class is the top of the food chain in OOP. The class is the blueprint of the data structure. The class defines the attributes of the data structure, as well as the methods, or actions associated with the data.

    • EX: To define a class for Movies the attributes, or instance variables, could be: movie_title, movie_storyline, movie_poster, movie_trailer, length_of_movie, etc. Within the class you can also define a method, which is a function within a class. In the Movie example a Show Trailer method could be established to open the trailer.
Objects/Instance
  • In OOP an object refers to a particulate instance of a class. In the class Movie example, an instance of the Movie class could be named Toy_Story. When the instance (object) is created it has to supply the necessary variable data defined in the class.
    • In the Movie example, you would enter the particulate movie_title (Toy Story), movie_storyline (A boy and his toys that come to life.), movie_poster (link to movie poster image), movie_trailer (link to movie trailer), length_of_movie (120 minutes), etc.
  • The benefit of using the class to define the variable structure is that you can create multiple instances of that class without reproducing the same defined variable code over and over.
Library
  • A library is a collection of programming modules used by a program. An example of libraries would be the Python Standard Library for Python language. This library houses all of the standard classes, objects, and functions for the Python language, as well as all documentation related to these.
  • The Python Standard Library
Module
  • A module is the file that houses functions from the Python Standard Library, or the file that houses the class and object files. When using a class or object from another file you first have to import the module name at the top of your code.
Constructor
  • The constructor is what initializes the new object, or instance, into memory. Within the constructor the instance variables are created to define an object.
  • The "def __init__:" function is used as a contructor in Python, and within this function is where instance varaibles are defined.
Self
  • Self is always the first parameter defined in the "__init__" function. Self refers to the name of the instance being created.
Instance Variables
  • Instance variables are the variables associated with a specific instance, and are defined in the "__init__" function within a class. Instance variables are accessed using the "self" keyword inside of the class, and with the instance name outside of the class.
Instance Methods
  • While instance variables define what data is in the class, the instance methods are functions within the class that define any action taken with the data.

    • EX: In the Movie class example, we defined "movie_trailer" as one of the variables. If we wanted to show that trailer when selected we could create a "show_trailer" method. You can see an example in the picture below.
Class Variable
  • A variable or constant within the class, but outside of the "__init__" function. This variable or object would apply to all instances of the class.
  • The standard is to show all class variables in UPPER CASE.

    • EX: In the Movie class example, we may want to create a class constant "VALID_RATINGS = ["G","PG","PG-13","R"]". This constant says that only instances of movies with these ratings could be created.
Inheritance
  • Inheritance allows new classes/objects to use instance variables from existing classes/objects. The class that is inherited from is called a parent class, while the new class that inherits from the parent class is called a child class. The child class can inherit all or some of the parent class attributes while also adding additional attributes of its own.
Method Overriding
  • Method overriding allows a child class to create a method that already exists in one of its parent classes. The child class could utilize the method found in the parent class, but it would only be able to utilize data found within the parent class.When the same method is created within the child class it then overrides the parent class method and can utilize all parent data, as well as data created within the child class.
Abstraction
  • By definition, abstraction is the process of taking away or removing characteristics from something in order to reduce it to a set of essential characteristics.
